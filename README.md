# 🛒 Amazon Frontend Clone

A responsive Amazon homepage clone built using **HTML** and **CSS**.  
This project focuses on replicating the UI design of Amazon, including layout, styling, and responsiveness.

---

## 🚀 Live Demo
🔗 amazon-clone-frontend-rust.vercel.app

---

## 📌 Features

- Amazon-like navigation bar
- Search bar UI
- Product categories section
- Banner section (hero image)
- Responsive layout
- Clean and structured design
- Hover effects for better UI experience

---

## 🛠️ Tech Stack

- HTML5
- CSS3
